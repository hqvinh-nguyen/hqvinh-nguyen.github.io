# Personal Portfolio

This is my personal portfolio website, built to present my learning path, projects, and technical direction in Mechatronics, Embedded Systems, and Automation.

## Purpose

The goal of this portfolio is not only to introduce myself, but also to create a clean place to organize my technical identity:

- who I am
- what I am learning
- what projects I am building
- how my skills are developing over time

## Tech Stack

- HTML
- CSS
- GitHub Pages

## Structure

The website is organized as a one-page portfolio:

- Hero section
- About section
- Featured project
- Skills section
- Contact section

CSS is used for layout, colors, spacing, responsive design, cards, buttons, and visual identity.

AI Assistance

This portfolio was refactored with AI assistance.

I used AI to save time on layout structure, CSS styling, spacing, color system, and responsive design. Instead of rewriting every line from scratch, I reviewed the generated code, studied how the sections were organized, and learned how each class and ID affects the layout.

AI helped accelerate the process, but I still focus on understanding:

how HTML sections are structured
how CSS classes target elements
how layout and alignment work
how colors and spacing create a professional visual style
how to modify the site by myself later
What I Learned

Through this portfolio, I practiced:

organizing a one-page website
using semantic HTML sections
linking navigation to section IDs
styling reusable components with CSS classes
building a cleaner and more professional online identity
using AI as a tool, not as a replacement for understanding
